# Project Conventions & Permanent Theme Instructions

## Theme & Visual Design
- **Theme Style**: Variation 2 (Dark Mode + Luxury Gold Accents).
- **Background Color**: Dark canvas (`#0d0d0f` / `#0a0a0c` / HSL `240 10% 4%`).
- **Primary / Accent Color**: Gold (`#FFD700` / HSL `48 100% 50%`).
- **Typography**: `Cairo`, `Syne`, `Geist Mono` / `Space Mono` for technical labels and numbers.
- **Borders & Panels**: High-contrast subtle borders (`rgba(255, 255, 255, 0.1)` / `border-white/10`) with dark semi-transparent glass/panel cards (`#111114` / `#161618`).
- **Layout Consistency**: Ensure both the Admin Dashboard (`/admin`) and Telegram Mini App (`/app`) strictly adhere to the dark gold palette without reverting to default light themes or unstyled templates.
