# MegaTurboEarn Platform

نظام متكامل يتكون من بوت تليجرام ولوحة تحكم ويب إدارية، مصمم للعمل مع Cloudflare.

## المكونات
1.  **لوحة التحكم (Web Admin)**: مبنية باستخدام React و Vite و @blinkdotnew/ui.
2.  **بوت تليجرام (Telegram Bot)**: مصمم كـ Cloudflare Worker باستخدام Hono.
3.  **قاعدة البيانات (Database)**: تستخدم Cloudflare D1 (SQLite).

## البدء السريع

### 1. إعداد قاعدة البيانات
قم بإنشاء قاعدة بيانات D1 في حسابك على Cloudflare وقم بتشغيل ملف `schema.sql` لإنشاء الجداول المطلوبة.

### 2. لوحة التحكم الإدارية
بيانات الدخول الافتراضية:
- **البريد الإلكتروني**: `admin@megaturbo.com`
- **كلمة المرور**: `admin123`

يمكنك تغيير هذه البيانات من صفحة الإعدادات بعد تسجيل الدخول.

### 3. بوت تليجرام
انتقل إلى مجلد `bot/` واتبع التعليمات الموجودة في `bot/README.md` لنشر البوت.

## المميزات
- تصميم عصري يدعم اللغة العربية بالكامل (RTL).
- ألوان احترافية (أزرق داكن وذهبي).
- نظام مستويات (برونزي، فضي، ذهبي، بلاتيني).
- نظام إحالات من مستويين مع عمولة خفية للمالك.
- إدارة كاملة للإعلانات والمهام وطلبات السحب.
- إحصائيات دقيقة ورسوم بيانية في لوحة التحكم.

## المتغيرات البيئية المطلوبة
- `BOT_TOKEN`: توكن البوت من BotFather.
- `ADMIN_ID`: معرف تليجرام الخاص بك ليتم منحك صلاحيات المدير.
